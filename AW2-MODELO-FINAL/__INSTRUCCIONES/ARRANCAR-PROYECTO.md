# Empezar a desarrollar

_Debe tener instalado en su computadora NodeJS version >=20 y NPM_

## Instalar paquetes configurados en package.json

Para correr el proyecto, debe instalar los paquetes.

1. En VSCODE -> Dentro de la carpeta `/proyecto`, hacer click derecho sobre `package.json` y abrirlo en consola. Esto garantiza que estamos en la ruta correcta.
2. En la consola correr el comando: `npm install`
3. Para desarrollo -> correr: `npm run dev`
4. Para Produccion y verificar -> correr: `npm start`
